# ✈️ Flight Reservation System

A full-stack **DBMS Mini Project** that simulates a real-world airline reservation platform.  
This project allows users to search flights, view bookings, check seat availability, monitor revenue, and manage passenger data through an interactive web dashboard.

---

## 🚀 Live Project Overview

The system is designed to demonstrate the practical implementation of:

- Database Management Systems (DBMS)
- Backend API Integration
- Frontend Web Development
- Real-time Data Fetching
- SQL Query Execution in Web Applications
